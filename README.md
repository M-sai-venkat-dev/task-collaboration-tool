## Overview

Task Collaboration Tool is a web application developed using React.js and Tailwind CSS that helps users create, assign, manage, and track tasks efficiently.

The application provides an intuitive interface for task management with features such as task assignment, priority management, task completion tracking, search functionality, and dashboard statistics.

---

## Features

* Create New Tasks
* Add Task Description
* Assign Tasks to Team Members
* Set Task Priority (Low, Medium, High)
* Search Tasks
* Mark Tasks as Completed
* Delete Tasks
* Dashboard Statistics
* Local Storage Data Persistence
* Responsive User Interface

## Usage

1. Enter task title and description.
2. Assign the task to a team member.
3. Select task priority.
4. Click **Add Task**.
5. View task statistics on the dashboard.
6. Search tasks using the search bar.
7. Mark tasks as completed when finished.
8. Delete tasks when no longer required.
